Aula - GIT HUB

Lista de comandos para GIT HUB

  • ls - LISTAR
  • pwd - onde estou?
  • cd - change directory
  • cd ~ - volta para a raiz
  • cd . - volta uma pasta acima
  • cd nome-da-pasta - para entrar em uma pasta
  • touch nome-do-arquivo.extensão - criar arquivo
  • mkdir nome-do-diretorio - cria diretório

Usando o git no terminal

  • git config --global user.name- “Ana Paula Rocha"
  • git config --global user.email- “apdsrocha@gmail.com"
  • git init - avisa o git para ficar de olho na pasta
  • git status - nos mostra o que foi modificado, adicionado, ou deletado da pasta
  • git add .- vamos colocar na caixinha todos os arquivos
  • git commit -m “inclui botão vermelho”- vamos colocar a etiqueta na caixinha
  • git log mostra a timeline do nosso projeto
  • git checkout -- arquivo.txt- para voltar para a versão do último commit
  • git branch nome-do-branch - para criar um branch
